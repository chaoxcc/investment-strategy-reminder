# 我的投资策略 · 最终执行版

> **生效日期**：2026-06-24  
> **核心理念**：用规则代替情绪，用纪律代替预测。  
> **设计目标**：消除所有模糊边界，可完全自动化执行，并内置防遗忘保障。  
> **适用周期**：未来6个月，每6个月复盘迭代一次。

---

## 一、经验与设计原则

本策略基于实战反馈反复打磨而成。以下经验已融入每一条规则：

- **只看指数，不看账户**：指数实时、客观，避免净值延迟和盈亏干扰。
- **滚动窗口代替自然周**：任何连续20个交易日区间，才是真正的趋势信号。
- **冷却期防止连锁反应**：触发后20个交易日暂停同类信号，避免过度交易。
- **卖出与定投完全分离**：卖出处理存量，定投处理增量，互不干扰。
- **大跌加仓是主动行为**：利用市场恐慌，而不是被恐慌利用。
- **美股卖出是趋势避险**：周线破位先保命，不猜顶。
- **资金回补必须有条件**：等止跌企稳再分批买回，拒绝冲动接飞刀。
- **策略进化靠复盘，不靠盘中**：任何修改只在周末冷静时进行。
- **消息越少，纪律越强**：只发行动指令，不发行情播报。
- **收到即执行，过期不候**：每个通知都有严格的操作窗口。

## 五、信号规则与边界条件

### 5.1 A股卖出信号

| 项目 | 内容 |
| :--- | :--- |
| **监控指标** | 创业板指 **滚动20个交易日** 累计涨幅 |
| **触发条件** | 滚动20日涨幅 **首次 > 8%** |
| **操作** | 赎回 融通创业板 + 广发中证500 各 **1/3** |
| **冷却期** | 触发后 **20个交易日** 内，A股卖出信号暂停，即使再次满足也忽略 |
| **冷却期结束后** | 重新进入监控，若滚动20日涨幅再次 > 8%，正常触发 |

**关键定义**：
- **滚动窗口**：以最近一个已收盘交易日（周日检查时为上周五）为终点，往前数20个交易日，计算区间涨幅。
- **首次触发**：当前滚动窗口内第一次突破阈值。冷却期内的突破不算“首次”，直接忽略。
- **信号消失**：若周日检查时，滚动20日涨幅已回落至8%以内，不触发。

### 5.2 美股卖出信号

| 项目 | 内容 |
| :--- | :--- |
| **监控指标** | 纳斯达克100 **滚动20个交易日** 累计跌幅 |
| **触发条件** | 滚动20日跌幅 **首次 > 5%** |
| **操作** | 赎回 嘉实美国成长 + 广发全球精选 各 **1/3** |
| **冷却期** | 触发后 **20个交易日** 内，美股卖出信号暂停 |
| **冷却期结束后** | 重新进入监控 |

> **注意**：这是趋势破位避险，不是止盈。主动基金重仓半导体/AI，跌幅易被放大，卖出1/3是为了防止泡沫破裂时的深度回撤。

### 5.3 A股加仓信号

| 项目 | 内容 |
| :--- | :--- |
| **监控指标** | 创业板指 **当日实时跌幅** |
| **触发条件** | 当日实时跌幅 **> 3%** 且 **本月加仓次数 < 2** |
| **操作** | 当天下午3点前，手动买入融通创业板 **1,000元** |
| **频率限制** | 每月最多 **2次**（每月1日重置计数） |

> **日内判断**：以14:00创业板指实时涨跌幅为准。若14:00满足，即使后续跌幅收窄，仍应执行加仓。  
> **加仓资金来源**：余额宝预备队。  
> **中证500和美股**：不设手动加仓，靠定投持续积累。

## 七、资金回补规则

### 7.1 A股赎回资金
- **不单独设回补规则**，通过 **定投 + 创业板大跌加仓** 自然回流市场。
- **严禁**在未触发加仓信号时，一次性将资金全部买回。

### 7.2 美股赎回资金
- **回补条件**：纳指100 **连续2周不创新低**（本周最低点 > 上周最低点）。
- **回补操作**：
  1. 条件满足后的下一周，将赎回资金的 **50%**，通过每日定投之外的方式分批买入（因有限额，需分多日）。
  2. 剩余 **50%**，等纳指100重回 **60日均线** 上方再买回。
- **通知方式**：美股赎回通知中附带此回补提醒。

## 八、定投纪律

1. **无论市场涨跌，定投绝不停止。**
2. 暂停定投的唯一条件：失业或急需用钱的重大意外。
3. 定投金额和日期，设好后永不修改（除非本策略版本升级）。

---

## 九、核心铁律（永不违反）

1. **卖出和定投互不干扰。** 同一天卖出，定投照常扣款。
2. **卖出和加仓互不干扰。** 同一天可同时执行。
3. **不因下跌暂停定投。** 下跌是定投的朋友。
4. **不在盘中临时修改规则。** 任何修改只在周末冷静时进行。
5. **通知即执行。** 收到通知在窗口内操作，没收到就什么也不做。
6. **错过窗口按补救规则执行，绝不追加。**
7. **极端情况兜底条款：** 若遇本策略未覆盖的极端市场情况（如连续涨跌停、交易所休市超一周），一律暂不操作，等待策略升级。临时决策不得违反以上铁律。
